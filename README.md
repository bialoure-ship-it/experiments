# Custom Agent Generator
*Brief sourced from a generator; solution and strategy by me.*

This agent creates fully documented, reusable operator-style agents.  
It is designed for experimentation, portfolio work, and rapid prototyping of new workflows.

---

## Purpose
The Custom Agent Generator produces new agents on demand.  
Each generated agent includes a clear role, behavior, constraints, inputs, outputs, and templates.

This allows you to:
- Prototype new workflows quickly  
- Maintain consistent structure across agents  
- Build a library of reusable operator tools  
- Document experiments in a repeatable format  

---

## How It Works
The agent asks 1–2 clarifying questions when needed, then outputs a complete agent specification.

Each generated agent includes:
- Purpose  
- Role  
- Behavior  
- Inputs required  
- Outputs  
- Templates  
- Example interaction  

---
